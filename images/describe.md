## Overall Design Direction

This website uses a **modern luxury wellness** style with a **minimalist editorial layout**.
The feeling is:

* Calm
* Premium
* Feminine but professional
* Spacious
* Soft wellness/lifestyle branding
* Scandinavian/Japanese-inspired minimalism

The design is strongly optimized for:

* Pilates / Yoga / Wellness studios
* Boutique fitness brands
* Lifestyle coaching
* Spa / rehabilitation services

# Typography

## Heading Font

Elegant serif font.

Looks similar to:

* Cormorant Garamond
* Playfair Display
* Libre Baskerville

Example:

```css
font-family: "Cormorant Garamond", serif;
```

Used for:

* Hero titles
* Section headings
* Emotional branding

---

## Body Font

Modern clean sans-serif.

Looks similar to:

* Inter
* Manrope
* Plus Jakarta Sans

Example:

```css

```

# UX Direction

The UX is focused on:

* emotional trust
* calm browsing
* premium experience
* low cognitive load

Not designed for:

* flashy interactions
* dense information
* technical dashboards

---

# Animation Suggestions

Use:

* fade-up on scroll
* slow hover zoom on images
* soft button hover
* opacity transitions

Avoid:

* bounce
* aggressive motion
* fast parallax

Suggested:

```css
transition: all 0.3s ease;
```
