# LabelLa Pilates Landing Page

A simple static landing page scaffold for a Pilates studio, inspired by the provided website design.

## What is included

- `index.html` — landing page structure
- `styles.css` — responsive styling and layout
- `script.js` — mobile menu toggle and smooth scrolling

## Usage

1. Open `index.html` directly in your browser
2. Or serve the folder with any static server

### Run locally with Python

```bash
cd /Users/trandinhhuy/PilateProject
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
